# Volunteer-Management-Platform


This project is a **Node.js backend application for managing volunteers**.  
It allows basic management of volunteers and events, with authentication and middleware for secure access.

## Features

- Add, login, edit, and remove volunteers
- Create and manage events
- Authentication using **tokens**
- Middleware for protecting routes
- Test database operations using **Postman**

## Technologies Used

- **Node.js** for the backend
- **Express.js** for building the API
- **MongoDB** for database management
- **JWT (JSON Web Tokens)** for authentication
- Middleware for route protection
- Environment configuration with **.env file**
